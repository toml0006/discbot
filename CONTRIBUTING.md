# Contributing to Discbot

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Open `discbot.xcodeproj` in Xcode
4. Make your changes
5. Test on macOS 15 (Sequoia) or earlier
6. Submit a pull request

## Guidelines

- Keep changes focused and minimal
- Test with actual hardware if possible
- Update documentation for user-facing changes
- Follow existing code style

## Reporting Issues

Please include:
- macOS version
- Changer model and connection type
- Steps to reproduce
- Any error messages

## Questions?

Open an issue for discussion.
